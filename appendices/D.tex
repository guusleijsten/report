.5 MCTargetDesc.
.6 SimdABIInfo.cpp\DTcomment{SIMD ...}.
.6 SimdABIInfo.h\DTcomment{SIMD ...}.
.6 SimdAsmBackend.cpp\DTcomment{SIMD ...}.
.6 SimdAsmBackend.h\DTcomment{SIMD ...}.
.6 SimdBaseInfo.h\DTcomment{SIMD ...}.
.6 SimdELFObjectWriter.cpp\DTcomment{Prints Object ELF}.
.6 SimdELFStreamer.h\DTcomment{Empty file, should be removed}.

.6 SimdMCTargetDesc.h\DTcomment{SIMD ...}.
.5 SimdAsmPrinter.cpp\DTcomment{Prints Machine Instrs to assembly file}.
.5 SimdAsmPrinter.h\DTcomment{Machine Instr Printer header file}.
.5 SimdBranchSimplify.cpp\DTcomment{Custom Pass to remove obsolete jump instructions}.

.5 SimdISelLowering.h\DTcomment{Lowering header file}.

.5 SimdVLIWPacketizer.h\DTcomment{SIMD Packetizer header file}.
